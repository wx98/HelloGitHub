# 标准 Markdown 语法

## 段落、标题、区块代码
### 段落
```
一个段落是由一个以上的连接的行句组成，而一个以上的空行则会划分出不同的段落，
想依赖Markdown插入<br/> ,在插入处先输入2个以上的空格再回车。
```
### 标题
标题语法有2种分别为：类Setext、类atx式   
#### 类Setext式:   
采用底线的形式，语法如下(= 最高阶标题、- 第二阶标题，且任何数量'=''-'都有效)   
__语法示例：__
```
一级标题
========
二级标题
--------
```
示例：
>标题 1
>========
>标题 2
>--------

#### 类atx式:   
在行首插入1到6个`#`分别对应`<h1>`~`<h6>`,  
并且你也可以为了美观把他们封闭起来,语法格式如下:   
__语法示例：__
```
# <h1>
## <h2>
### <h3>
#### <h4> ####
##### <h5> #####
###### <h6> ######
```
示例:
># `<h1>`
>## `<h2>`
>### `<h3>`
>#### `<h4>` ####
>##### `<h5>` #####
>###### `<h6>` ######

### 区块的引用
在每一行前加上`>`以达成引用的目的   
__语法示例：__
```
>    这是段用来演示的段落,在此段
>落的每一行行首加上尖括号'>'以达
>到引用的目的。
>
>明白了吗？
```
示例：
>    这是段用来演示的段落,在此段   
>落的每一行行首加上尖括号'>'以达  
>到引用的目的。   
>  
>明白了吗？   

当然Markdown也允许你只给每一段首行加入`>`   
__语法示例：__
```
>    这是段用来演示的段落,在此段   
落的每一行行首加上尖括号'>'以达   
到引用的目的。   

>明白了吗？  
```
示例：
>    这是段用来演示的段落,在此段   
落的每一行行首加上尖括号'>'以达   
到引用的目的。   
>   
>明白了吗？   

区块之间还可以进行嵌套   
__语法示例：__
```
>    这是段用来演示的段落,在此段   
>落的每一行行首加上尖括号'>'以达   
>到引用的目的。    
>>大概语法像这样`>例子`   
>
>明白了吗？
```
示例：
>    这是段用来演示的段落,在此段   
>落的每一行行首加上尖括号'>'以达   
>到引用的目的。   
>>大概语法像这样`>例子`   
>
>明白了吗？   

区块内也可以其他的 Markdown 语法，包括标题、列表、代码区块等   
__语法示例：__
```
>#### 这是一个标题。
>
> 1.   这是第一行列表项。
> 2.   这是第二行列表项。
> >*斜斜斜斜斜*   **粗粗粗粗粗**

>
>     pritf("Hello would!");
```
示例：
>#### 这是一个标题。
>
> 1.   这是第一行列表项。   
> 2.   这是第二行列表项。   
> >*斜斜斜斜斜*
>
>  **粗粗粗粗粗**   
> 给出一些例子代码：  
>
>     pritf("Hello would!");

### 列表
Markdown 有序列表和无序列表2种形式
#### 无序列表使用星号、加号或是减号
__语法示例：__
```
* 1
* 2
* 3

+ 4
+ 5
+ 6

- 7
- 8
- 9
```
示例：
>* 1
>* 2
>* 3
>
>+ 4
>+ 5
>+ 6
>
>- 7
>- 8
>- 9

#### 有序列表则使用数字接着一个英文句点  
__语法示例：__
```
1.  Bird
2.  McHale
3.  Parish
```
示例：
>1.  Bird
>2.  McHale
>3.  Parish

请注意的一点是.标记上的数字对于并不会影响Html的输出结果   
比如你写成
```   
1.  Bird
1.  McHale
1.  Parish
或者
3.  Bird
9.  McHale
6.  Parish
```
示例：
>1.  Bird
>1.  McHale
>1.  Parish
>>或者
>
>3.  Bird
>9.  McHale
>6.  Parish



## 1.斜体和加粗
```
*斜斜斜斜斜*   **粗粗粗粗粗**
_斜斜斜斜斜_   __粗粗粗粗粗__
```
示例：   
>*斜斜斜斜斜*   **粗粗粗粗粗**   
_斜斜斜斜斜_   __粗粗粗粗粗__

## 5.清单目录
### 5.1有序，没有段落:
```
1.  Foo
2.  Bar
```
示例：
1.  Foo
2.  Bar

### 5.2无序，有段落：
```
*   A list item.

    With multiple paragraphs.

*   Bar
```
示例：
*   A list item.

    With multiple paragraphs.

*   Bar
### 5.3可以嵌套
```
*   Abacus
    * answer
*   Bubbles
    1.  bunk
    2.  bupkis
        * BELITTLER
    3. burper
*   Cunning
```
示例：

*   Bubbles
